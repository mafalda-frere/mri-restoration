# Deep Learning for MRI reconstruction: about hallucinations and instabilities

This project explores **MRI image reconstruction from undersampled k-space data using deep learning**.  
It reproduces experiments illustrating the **accuracy–stability trade-off and hallucination effects in inverse problems**, based on the work of **Gottschling et al. (2025)**.

The tutorial implements and studies reconstruction models introduced in:
- **AUTOMAP** (Zhu et al., 2018)
- **Deep Cascade CNN for MRI reconstruction** (Schlemper et al., 2017)

Experiments use data adapted from the **NYU fastMRI dataset** (Knoll et al., 2020).

This work was completed as part of a coursework assignment for the **Deep Learning for Image Analysis** course at **AgroParisTech**, taught by **Romain Thoreau**.
